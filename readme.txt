== Description ==

= The best free plugin to display the Royal Mail live rates =

Use this free Royal Mail WooCommerce Live Rates plugin to offer your customers the Royal Mail shipping methods for domestic and international shipping.

**Don't waste your time** and **set everything up hassle free in your shop in just 5 minutes**!

Show your clients the Royal Mail shipping services with their prices being dynamically calculated in the cart and checkout. Once the Royal Mail Live Rates shipping method is added within a shipping zone, the **shipping cost for each Royal Mail service is calculated real-time** based on the products' weight, shop location and delivery destination.

= Features =

* Automatic shipping cost calculation and displaying the Royal Mail live rates in the cart and checkout
* Domestic and international Royal Mail services
* Royal Mail services' shipping cost calculation based on the cart weight and the shipping destination
* Limiting services only for those available for the customer's address
* Manual Royal Mail services limiting and possibility to choose which services should be displayed and which not
* Possibility to set the fixed fallback amount in case no Royal Mail rates were returned
* Free shipping over amount threshold
* Package default dimensions and weight
* Possibility to Include the optional compensation / insurance
* Optional excluding the tax from Royal Mail shipping rates
* Dedicated debug mode for easy troubleshooting

= PRO Features =

* Automatic multiple products box packing algorithm based on their weight and volume
* Possibility to define the custom boxes used for shipping and specify their size, weight and padding
* Fixed and percentage handling fees or discounts for Royal Mail rates
* Multi-currency support

== Installation	 ==

This plugin can be easily installed like any other WordPress integration by following the steps below:

1. Download and unzip the latest zip file release.
2. Upload the entire plugin directory to your /wp-content/plugins/ path.
3. Activate the plugin using the Plugins menu in WordPress sidebar menu.

Optionally you can also try to upload the plugin zip file using Plugins -> Add New -> Upload Plugin option from the WordPress sidebar menu. Then go directly to point 3.

== Frequently Asked Questions ==

= How can I configure Royal Mail services? =
To determine which Royal Mail services should be available for the customers to choose, use the **Services: Enable the services' custom settings** option and select from the **Services Table** the ones you want to be displayed.
